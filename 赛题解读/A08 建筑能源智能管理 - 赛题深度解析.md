# A08 建筑能源智能管理 - 赛题深度解读

> 出题单位：中建八局第二建设有限公司 | 纯软件

---

## 一、赛题核心解读

### 1.1 一句话总结

**A08 是做什么的？**

开发一个**建筑能源智能管理系统**，实现**能耗数据查询统计** + **大模型智慧运维问答**，帮助建筑管理者快速了解能耗情况、诊断异常原因。

---

### 1.2 业务场景举例

```
场景 1：物业经理查询能耗
- 问题："3 号楼上周的用电量是多少？比前周增长了吗？"
- 系统：自动查询数据库→生成对比图表→回答"用电量 1200kWh，增长 15%"

场景 2：运维人员故障诊断
- 问题："空调系统 COP 值偏低，可能是什么原因？"
- 系统：检索知识库→分析历史数据→回答"可能原因：1.过滤器堵塞 2.冷媒不足..."

场景 3：自动生成报表
- 操作：点击"生成月度报表"
- 系统：自动汇总数据→生成图表→导出 PDF/Excel
```

#### 数据量要求

- **最少**：1000 条记录
- **建议**：5000-10000 条（覆盖多建筑、多时段）
- **数据来源**：模拟生成（无需真实采集）

#### 数据生成方案

```python
# 示例：用 Python 模拟生成能耗数据
import pandas as pd
import numpy as np
from datetime import datetime, timedelta

def generate_energy_data(num_buildings=5, days=30):
    data = []
    for building_id in range(1, num_buildings + 1):
        for day in range(days):
            for hour in range(24):
                # 模拟小时级数据
                record = {
                    'building_id': f'B{building_id:03d}',
                    'building_type': np.random.choice(['办公楼', '教学楼', '宿舍']),
                    'timestamp': datetime(2026, 1, 1) + timedelta(days=day, hours=hour),
                    'electricity': np.random.uniform(50, 200),  # kWh
                    'ac_energy': np.random.uniform(30, 150),
                    'outlet_temp': np.random.uniform(10, 15),
                    'return_temp': np.random.uniform(15, 20),
                    'env_temp': np.random.uniform(20, 35),
                    'humidity': np.random.uniform(40, 80),
                    'device_id': f'EQ{building_id:03d}',
                    'status': np.random.choice(['正常', '正常', '正常', '异常'])
                }
                data.append(record)
    return pd.DataFrame(data)

# 生成并保存
df = generate_energy_data()
df.to_csv('energy_data.csv', index=False, encoding='utf-8-sig')
```

### 3.2 查询统计模块

#### 必须实现的 3 类统计分析

| 分析类型 | 功能说明 | 技术实现 |
|----------|----------|----------|
| **时段汇总** | 按日/周/月汇总能耗 | GROUP BY + SUM |
| **COP 计算** | 空调系统能效比计算 | COP = 制冷量/耗电量 |
| **数据异常分析** | 识别异常能耗数据 | 阈值判断/统计方法 |

#### COP 计算公式

```
COP（能效比）= 制冷量 / 空调耗电量

其中：
制冷量 ≈ 水流量 × 水的比热容 × (回水温度 - 出水温度)

简化计算（假设水流量恒定）：
COP ∝ (回水温度 - 出水温度) / 空调耗电量
```
